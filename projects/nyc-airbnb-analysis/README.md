# NYC Airbnb Analysis

## Overview

Exploratory data analysis of the NYC Airbnb dataset to understand patterns in pricing, availability, and location.

## Objectives

- Clean and explore Airbnb listing data
- Analyze price distributions
- Investigate neighborhood patterns
- Explore spatial relationships between listings
- Build predictive models for price estimation

## Dataset

NYC Airbnb Open Data.

## Tools

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- GeoPandas
- Scikit-learn

## Project Structure

- `data/raw` - original dataset
- `data/processed` - cleaned datasets
- `notebooks` - analysis workflow
- `src` - reusable Python functions
- `images` - visualizations
